üöÄ QUICK START (5 MINUTES):
================================================================================

On your Raspberry Pi:

1. Download git-sync-complete.tar.gz to your Pi

2. Extract:
   $ tar -xzf git-sync-complete.tar.gz
   $ cd git-sync

3. Run setup:
   $ chmod +x scripts/setup.sh
   $ ./scripts/setup.sh

4. Edit repo list:
   $ nano ~/sync-repos.json
   
   Add your repos:
   [
     {"name": "repo1"},
     {"name": "repo2"}
   ]

5. Update username:
   $ nano /opt/git-sync/sync.py
   
   Change: "github_username": "YOUR_GITHUB_USERNAME"

6. Test:
   $ python3 /opt/git-sync/sync.py

7. Check status:
   $ git-sync-status.sh

DONE! Your repos will sync automatically every hour.

üîë GITHUB TOKEN:
================================================================================

Get your token at: https://github.com/settings/tokens

Required:
   - Token type: "Personal access token (classic)"
   - Scope: "repo" (full control of private repositories)
   - Expiration: Your choice (or "No expiration")

The setup script will prompt for your token.

================================================================================
‚úÖ WHAT GETS INSTALLED:
================================================================================

Scripts:
   /opt/git-sync/sync.py - Main sync script
   /usr/local/bin/git-sync-status.sh - Status command

Configuration:
   ~/sync-repos.json - List of repos to sync
   /etc/systemd/system/git-sync.env - GitHub token (secure)

Repositories:
   ~/repositories/ - Bare Git repos (server storage)
   ~/sync-workspace/ - Mirror repos (for syncing)

Services:
   /etc/systemd/system/git-sync.service - Sync service
   /etc/systemd/system/git-sync.timer - Hourly timer

================================================================================
üîß COMMON COMMANDS:
================================================================================

Status:
   $ git-sync-status.sh

Manual Sync:
   $ sudo systemctl start git-sync.service

View Logs:
   $ sudo journalctl -u git-sync.service -f

Service Control:
   $ sudo systemctl start/stop/restart git-sync.timer
   $ sudo systemctl enable/disable git-sync.timer

Clone from Pi:
   $ git clone user@pi-ip:repositories/repo-name.git

================================================================================
üÜò TROUBLESHOOTING:
================================================================================

Service not running:
   $ sudo systemctl status git-sync.service
   $ sudo journalctl -u git-sync.service -n 50

Authentication failed:
   $ sudo nano /etc/systemd/system/git-sync.env
   (Update token, then restart service)

Permission errors:
   $ sudo chown -R $USER:$USER ~/repositories
   $ sudo chown -R $USER:$USER ~/sync-workspace
